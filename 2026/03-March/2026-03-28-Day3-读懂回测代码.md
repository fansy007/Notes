def load_config():

### operators.py — 基础运算符

打开文件：`E:\workspace\EasyXT\101factor\101factor_platform\src\factor_engine\operators.py`

这些是因子计算的基础工具函数：

```python
# 移动平均
def sma(df, window=10, weight=1):
    """简单移动平均"""
    return df.rolling(window=window).mean()

# 排名
def rank(df):
    """横截面排名（同一时刻对所有股票排名）"""
    return df.rank(axis=1)

# 差分
def delta(df, period=1):
    """时间差分，相当于 Java: df[i] - df[i-period]"""
    return df.diff(period)

# 延迟
def delay(df, period=1):
    """时间延迟，相当于 Java: df[i-period]"""
    return df.shift(period)

# 最大值
def ts_max(df, window):
    """滚动窗口最大值"""
    return df.rolling(window=window).max()

# 最小值
def ts_min(df, window):
    """滚动窗口最小值"""
    return df.rolling(window=window).min()

# 相关系数
def correlation(x, y, window):
    """滚动窗口相关系数"""
    return x.rolling(window).corr(y)
```

**量化概念：**
- `delta(close, 1)` = 今日收盘价 - 昨日收盘价
- `delay(close, 1)` = 昨日收盘价
- `rank(close)` = 在某一天，按收盘价排序所有股票

### alpha101.py — 因子实现

打开文件：`E:\workspace\EasyXT\101factor\101factor_platform\src\factor_engine\alpha101.py`

#### alpha002 — 量价相关性

```python
def alpha002(self):
    """
    公式: -1 * correlation(rank(delta(log(volume), 2)), rank((close-open)/open), 6)
    
    含义:
    1. delta(log(volume), 2) = 成交量的2日变化率
    2. rank(...) = 横截面排名（值越大排名越高）
    3. correlation(..., 6) = 6日滚动相关系数
    4. 取负号 = 负相关更好
    
    解读:
    - 放量上涨（正相关）-> 因子值负 -> 不选
    - 放量下跌（负相关）-> 因子值正 -> 选
    """
    return -1 * correlation(
        rank(delta(np.log(self.volume), 2)),
        rank((self.close - self.open) / self.open),
        6
    )
```

#### alpha080 — 成交量变化率

```python
def alpha080(self):
    """
    公式: (volume - delay(volume, 5)) / delay(volume, 5) * 100
    
    含义:
    - 5日成交量变化率（百分比）
    - 正值 = 放量
    - 负值 = 缩量
    
    解读:
    - 量能放大 = 有资金关注
    - 如果因子方向是升序（逆向），选负值最大的 = 放量最明显的
    """
    return (self.volume - delay(self.volume, 5)) / delay(self.volume, 5) * 100
```

## buy_factor_strategy.py — 选股策略

### 核心选股逻辑

```python
def select_stocks(factor_results, config):
    """
    选股流程:
    1. 提取每个因子的最新值
    2. 对每个因子排名
    3. 加权求和
    4. 排序取前N名
    """
    
    # 1. 提取最新因子值
    factor_values = {}
    for factor_name in config['factors']:
        factor_df = factor_results[factor_name]
        latest_date = factor_df.index.get_level_values(0)[-1]
        factor_values[factor_name] = factor_df.loc[latest_date]
    
    # 2. 归一化因子值（按绝对值排名）
    # 关键代码：
    ranks = abs(factor_values).rank(ascending=False)
    # -60.86 排名1 -> 得分最高
    # -45.95 排名4 -> 得分较低
    
    # 3. 计算综合得分
    weights = config['factor_weights']
    composite_score = sum(ranks[f] * w for f, w in zip(config['factors'], weights))
    
    # 4. 排序取前N
    selected = composite_score.sort_values(ascending=False).head(config['top_n'])
    
    return selected.index.tolist()
```

### factor_reverse 参数详解

| factor_reverse | 效果 | 适用场景 |
|----------------|------|----------|
| `false` | 选正值最大的 | 因子本身正值好 |
| `true` | 选负值最大的（逆向） | 量价因子：负值=放量上涨=好 |

**为什么量价因子用 `factor_reverse=true`？**

```python
# alpha080 成交量变化率
# 放量 -> 负值（比如-60表示放量60%）
# 缩量 -> 正值（比如+20表示缩量20%）

# 放量上涨是好事！
# 但 alpha080 是负值

# factor_reverse=true -> 选负值最大的 -> 选放量最明显的
# 所以量价因子要用逆向！
```
