# CS500极简市场评分 - 每日5分钟

## 🎯 为什么用CS500判断市场大势？

**中证500指数的核心价值：**
- 代表**中小盘股**（剔除沪深300后的500只股票）
- 对**市场流动性**和**情绪变化**敏感
- 通常是**趋势的先行指标**（比上证指数更早反应）
- 能反映**成长性**和**风险偏好**

## 📱 快速打分表（今日：2026-03-13）

### 第一步：数据快速获取（2分钟）
1. **打开东方财富APP/网站**
2. **搜索"中证500"**
3. **记录以下数据：**
   ```
   今日收盘价：______ 点
   市盈率PE：______ 倍
   十年历史分位点：
   涨跌幅：______ %
   成交量：______ 亿股
   ```

### 第二步：温度计打分（1分钟）

**PE感觉（对比最近1年）**
- [ ] **很便宜**（PE处于最低30%区域） → **+2分**
- [ ] **正常价**（PE中间40%区域） → **+0分**
- [x] **有点贵**（PE最高30%区域） → **-2分**

**成交量感觉（对比前5天平均）**
- [ ] **很活跃**（明显放量） → **+2分**
- [x] **正常**（差不多） → **+0分**
- [ ] **冷清**（缩量） → **-2分**

### 第三步：趋势判断（1分钟）

周级别表现
- [ ] 向上（涨幅超过沪深300） → **+2分**
- [x] **横盘**（涨跌相近） → **+0分**
- [ ] **向下**（涨幅落后沪深300） → **-2分**

**尾盘表现（2:30-3:00）**
- [ ] **拉升** → **+2分**
- [x] **横盘** → **+0分**  
- [ ] **跳水** → **-2分**

### 第四步：情绪判断（1分钟）

**你周围的讨论热度**
- [ ] **没人聊** → **+2分**（观察者阶段）
- [x] **正常聊** → **+0分**（平衡阶段）
- [ ] **热聊** → **-2分**（跟随者阶段）

**你自己的心态**
- [ ] **有点怕**（想卖不敢买） → **+2分**
- [x] **平常心** → **+0分**
- [ ] **很兴奋**（怕踏空想冲） → **-2分**

## ⚡ 计算今日分数

把你选择的分数加起来：
```
温度计分：_0__（PE感觉 + 成交量感觉）
趋势判断分：__-3_（相对表现 + 尾盘表现）
情绪判断分：___0（讨论热度 + 自己心态）
--------------
总分：_-3__ 分
```

## 📈 趋势预测信号

根据**CS500总分**判断市场整体趋势：
```
🔥 强烈看涨：8分以上
    → 市场情绪低，估值便宜，趋势向上
    → 大胆增加中小盘股仓位

↗️ 温和看涨：3-8分
    → 市场正常，有上涨动能
    → 可适当配置，精选个股

↔️ 中性震荡：-2到2分
    → 方向不明，可能震荡
    → 保持现有仓位，等待更明确信号

↘️ 温和看跌：-8到-3分
    → 市场偏热，有调整压力
    → 减仓高估值股，转向防御

💧 强烈看跌：-8分以下
    → 市场过热，风险较大
    → 大幅降低仓位，持币观望
```

1. **每天收盘后花5分钟**填写

4. **优化调整**：根据自己的经验微调评分标准
