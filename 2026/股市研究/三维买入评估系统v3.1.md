# 三维买入评估系统 v3.1（最终版）

> 注：本系统仅用于买入决策，不考虑卖出。
> v3.1 核心升级：删掉趋势维度，加入 CS500 市场环境过滤

---

## 🎯 系统目的
防止盲目追高、情绪化交易，建立可量化、可重复、可回测的买入决策框架。

---

## 📊 系统架构

### 前置开关：CS500 市场环境

| CS500评分 | 市场状态 | 操作 |
|-----------|----------|------|
| **≤7分** | 冷/转暖/正常 | 执行三维买入评估 |
| **>7分** | 热/极热 | 暂停买入，只持有 |

---

### 两个评估维度

| 维度 | 权重 |
|------|------|
| **量价形态** | **60%** |
| **基本面** | **40%** |

---

## 📊 一、量价形态（60%权重）

### 核心逻辑
放量上涨后缩量回调到支撑位，是最佳买点形态

### 信号条件（必须全部满足）

```
vol_ratio < 0.5      # 成交量 < 10日峰值50%（缩量）
drawdown < -10       # 价格从10日高点回撤 > 10%
close < ma20         # 收盘价 < 20日均线（短期走弱）
pct_change < 0       # 当日下跌
ma20_slope > 0       # 20日均线向上（趋势未破坏）
```

### 量化评分

| 条件                               | 得分    |
| -------------------------------- | ----- |
| 成交量 < 10日峰值30%，且价格回撤>15%，且MA20向上 | 9-10分 |
| 成交量 < 10日峰值50%，且价格回撤>10%，且MA20向上 | 7-8分  |
| 成交量 < 10日峰值70%，且价格回撤>5%          | 5-6分  |
| 成交量正常（70-100%峰值）                 | 3-4分  |
| 成交量放大（>100%峰值），且价格上涨             | 1-2分  |

---

## 📊 二、基本面（40%权重）

### 量化标准
最近两期财报的营收和净利润同比增速

| 条件 | 得分 |
|------|------|
| 营收+净利润双增长，增速>20% | 9-10分 |
| 营收+净利润双增长，增速5-20% | 7-8分 |
| 营收或净利润其中一项增长 | 5-6分 |
| 营收+净利润均低个位数增长或持平 | 3-4分 |
| 营收或净利润出现下滑 | 1-2分 |

---

## 🧮 计算方法

### 综合评分

```
综合分 = (形态分 × 0.6) + (基本面分 × 0.4)
```

### 操作决策矩阵

| 综合评分 | 建议操作 | 仓位建议 |
|----------|----------|----------|
| **8.0-10.0分** | **强烈买入** | 可重仓（50%以上） |
| **7.0-7.9分** | **正常买入** | 正常仓位（30-40%总资金） |
| **6.0-6.9分** | **轻仓试错** | 轻仓（≤20%总资金） |
| **4.0-5.9分** | **中性观望** | 不操作 |
| **0.0-3.9分** | **不买入** | 不考虑入场 |

---

## 🔄 使用流程

1. **查 CS500 评分** → 判断市场环境
   - 如果 >7分：暂停买入，只持有现有仓位
   
2. **定量价形态** → 形态分
   
3. **查基本面** → 基本面分
   
4. **计算综合分** → 确定仓位

---

## ⚠️ 已知局限性

1. **权重未经回测验证**：60%:40% 是初步设定，需更多数据验证
2. **CS500评分阈值7分**：待更多数据验证是否合理
3. **缺卖出系统**：本系统只管买入，卖出纪律需另行建立
4. **样本量不足**：目前仅4个历史信号，统计显著性有限
