# 市场评分极简版 - 每日3分钟

## 📅 2026-03-13 市场打分

### 第一步：温度判断（温度计）
请根据感觉选择：

**市盈率位置感觉**（客观数据可核对）
- [ ] **冷**（感觉处于历史低位） → **+3分**
- [ ] **温**（感觉正常水平） → **+1分**  
- [ ] **热**（感觉历史高位） → **-2分**

**媒体报道热度**（主观判断）
- [ ] **安静**（很少报道） → **+2分**
- [ ] **正常**（有报道但不多） → **+1分**
- [ ] **刷屏**（到处都在讨论） → **-3分**

### 第二步：HS模型判断（观察者vs跟随者）

**信号特征**（哪个更像当下？）
- [ ] **观察者阶段**（偷偷涨，朋友圈没人聊） → **+2分**
- [ ] **过渡阶段**（有涨有聊，还没热） → **+1分**
- [ ] **跟随者阶段**（大热点，股神辈出） → **-3分**

**尾盘表现**（今日下午2:30-3:00）
- [ ] **拉升**（最后半小时涨） → **+2分**
- [ ] **平稳**（横盘到尾） → **+1分**
- [ ] **跳水**（最后跌） → **-2分**

### 第三步：风险感觉（你的直觉）

**恐惧/贪婪指数**（你的内心）
- [ ] **恐惧**（怕跌不敢买） → **+2分**
- [ ] **淡定**（平常心） → **+1分**
- [ ] **贪婪**（怕踏空想冲） → **-3分**

### 第四步：计算总分

把你勾选的分数加起来：
```
温度分：___ 分
HS模型分：___ 分  
风险感觉分：___ 分
--------------
总分：___ 分
```

### 第五步：操作建议

根据总分：
- **8分以上**：大胆建仓（机会难得）
- **3-8分**：小仓试探（机会不错）
- **-2到2分**：持有观察（中性状态）
- **-2分以下**：考虑减仓（风险增加）
