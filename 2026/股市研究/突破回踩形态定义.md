## 形态逻辑

放量突破关键位置后，缩量回踩确认支撑位，形成"突破-回踩-再启动"的结构。

**核心思路**：突破是真突破还是假突破，用回踩的缩量程度来确认。

---

## 量化条件

### 第一步：识别突破日

```
条件A：成交量 > 20日均量的2倍
条件B：收盘价 > 20日最高价
同时满足 = 突破日
```

### 第二步：识别回踩

```
突破日后的第3-10天内：
条件C：成交量 < 突破日成交量的50%
条件D：收盘价在突破位附近（±5%以内）
同时满足 = 回踩确认
```

### 第三步：买入信号

```
回踩日后出现以下任一信号：
- 放量上涨（成交量 > 5日均量1.5倍）
- 阳包阴（当日收盘 > 前一日开盘）
- 突破5日均线
```

---

## 评分规则

| 条件 | 形态分 |
|------|--------|
| 突破后3-5天内回踩 + 缩量<30% + 再启动 | 9-10分 |
| 突破后3-10天内回踩 + 缩量<50% + 再启动 | 7-8分 |
| 突破后回踩 + 缩量<70% | 5-6分 |
| 回踩缩量不明显（>70%） | 3-4分 |
| 放量跌破突破位（假突破） | 1-2分 |

## 代码实现

```python
import pandas as pd
import numpy as np

def find_breakout(df):
    """
    寻找突破日
    df 需要包含: open, high, low, close, volume
    """
    df = df.copy()
    df['vol_ma20'] = df['volume'].rolling(20).mean()
    df['high_20d'] = df['high'].rolling(20).max()
    
    # 突破条件
    df['breakout'] = (df['volume'] > df['vol_ma20'] * 2) & \
                      (df['close'] > df['high_20d'].shift(1))
    
    return df[df['breakout']]

def find_pullback(df, breakout_idx, days_range=(3, 10)):
    """
    寻找回踩日
    breakout_idx: 突破日的索引
    days_range: 回踩窗口范围
    """
    breakout_vol = df.loc[breakout_idx, 'volume']
    breakout_price = df.loc[breakout_idx, 'close']
    
    # 在突破后3-10天内寻找回踩
    start_idx = breakout_idx + days_range[0]
    end_idx = breakout_idx + days_range[1]
    
    candidate = df.loc[start_idx:end_idx]
    
    # 回踩条件
    pullback = candidate[
        (candidate['volume'] < breakout_vol * 0.5) &  # 缩量
        (abs(candidate['close'] - breakout_price) / breakout_price < 0.05)  # 价格在突破位附近
    ]
    
    return pullback

def calculate_breakout_score(df, date):
    """
    计算某一天的突破回踩得分
    """
    df = df.copy()
    df['vol_ma20'] = df['volume'].rolling(20).mean()
    df['vol_ma5'] = df['volume'].rolling(5).mean()
    df['high_20d'] = df['high'].rolling(20).max()
    df['close_ma5'] = df['close'].rolling(5).mean()
    df['pct_change'] = df['close'].pct_change() * 100
    
    # 查找最近20天内的突破日
    idx = df.index.get_loc(date)
    recent = df.iloc[max(0, idx-20):idx]
    
    # 简化判断：最近10天内有放量突破，且当前处于缩量回踩状态
    vol_peak = recent['volume'].max()
    high_peak = recent['high'].max()
    
    current = df.iloc[idx]
    
    # 判断是否在回踩区间
    in_pullback = (current['volume'] < vol_peak * 0.5) and \
                  (current['close'] > high_peak * 0.95) and \
                  (current['close'] < high_peak)
    
    # 判断是否再启动
    resumption = (current['volume'] > current['vol_ma5'] * 1.3) and \
                 (current['close'] > current['close_ma5'])
    
    if in_pullback and resumption:
        return 9
    elif in_pullback:
        return 7
    elif current['volume'] > vol_peak * 0.7:
        return 4
    else:
        return 5
```

## 与缩量回调对比

| 维度 | 缩量回调 | 突破回踩 |
|------|----------|----------|
| 适用场景 | 下跌趋势中的反弹 | 震荡/底部突破后 |
| 信号频率 | 较少 | 适中 |
| 确定性 | 中 | 高（突破后确认） |
| 风险 | 可能是下跌中继 | 假突破 |
