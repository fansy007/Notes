# 顺网科技（300113）五维买入评估系统

> 注：本系统仅用于买入决策，不考虑卖出。

## 🎯 系统目的
防止盲目追高、情绪化交易，建立量化买入决策框架。

## 📊 五个评估维度（各20%权重）

### 1. 趋势健康度
**问题**：周线和月线趋势是否向上？
```
10分：周线月线明显向上，放量突破
7-9分：周线向上，月线平稳
4-6分：周线月线横盘震荡
1-3分：周线或月线明显走弱
```

### 2. 基本面稳健度
**问题**：公司基本面、题材热度是否健康？
```
10分：业绩增长+热门题材+行业龙头
7-9分：基本面稳定+题材有故事
4-6分：基本面平稳，无明显亮点
1-3分：基本面有隐忧（业绩下滑、竞争加剧）
```

### 3. 情绪极端度 ⚠️ 逆向思维！
**问题**：市场情绪是否到了极端位置？
```
9-10分：极度恐慌（暴跌、所有人看空）→ **最佳买入机会**
7-8分：较为恐慌 → **较好买入机会**
5-6分：情绪平稳/分歧 → **中性**
3-4分：较为乐观 → **警惕追高**
1-2分：极度贪婪（暴涨、所有人看多）→ **卖出时机**
```

**重要**：高分是买点，低分是卖点！

### 4. 价格吸引力
**问题**：当前价格相对于价值是否便宜？
```
10分：价格严重低估（感觉特别便宜）
7-9分：价格合理偏低
5-6分：价格合理
3-4分：价格偏高
1-2分：价格严重高估（感觉泡沫明显）
```

### 5. CS500环境分
**问题**：大盘整体环境如何？
```
转换公式：CS500_10分制 = (CS500_-10到+10分 + 10) ÷ 2

示例：
- CS500评分为 -10分 → 0分
- CS500评分为   0分 → 5分
- CS500评分为 +10分 → 10分
- CS500评分为  -3分 → 3.5分 ≈ 4分
```

## 🧮 计算方法

### 综合评分
```
综合分 = (趋势健康度 + 基本面稳健度 + 情绪极端度 + 价格吸引力 + CS500环境分) ÷ 5
```

### 操作决策矩阵

| 综合评分 | 建议操作 | 仓位建议 |
|----------|----------|----------|
| **8.0-10.0分** | **强烈买入** | 可重仓（突破重仓线） |
| **7.0-7.9分** | **正常买入** | 正常仓位（30-40%总资金） |
| **6.0-6.9分** | **轻仓试错** | 轻仓（20%以内总资金） |
| **4.0-5.9分** | **中性观望** | 不操作 |
| **0.0-3.9分** | **不买入** | 不考虑入场 |

## 🔄 使用流程

### 买入前评估（约5分钟）
1. **获取数据**：收盘后查看顺网科技价格、成交量、CS500数据
2. **五维度主观打分**（根据你的感觉和经验）
3. **计算综合评分**（简单平均）
4. **对照决策矩阵**确定操作建议和仓位
5. **记录**：保存评分和决策

### 特殊规则
- **只在考虑买入时评估**：不用于持续监控或卖出决策
- **一致性原则**：坚持使用，不因单次结果放弃系统
- **历史回测**：每周复盘评分准确性，优化标准
