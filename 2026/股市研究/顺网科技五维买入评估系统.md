# 三维买入评估系统 v3.0

> 注：本系统仅用于买入决策，不考虑卖出。
> v3.0 核心升级：简化维度，突出核心信号，基于"缩量回调"形态做决策

---

## 🎯 系统目的
防止盲目追高、情绪化交易，建立可量化、可重复、可回测的买入决策框架。

---

## 📊 三个评估维度

### 1. 量价形态（50%权重）— 核心信号
**核心逻辑**：放量上涨后缩量回调到支撑位，是最佳买点形态

**量化标准**：

| 条件 | 得分 |
|------|------|
| 成交量 < 10日峰值30%，且价格回撤>15%，且MA20向上 | 9-10分 |
| 成交量 < 10日峰值50%，且价格回撤>10%，且MA20向上 | 7-8分 |
| 成交量 < 10日峰值70%，且价格回撤>5% | 5-6分 |
| 成交量正常（70-100%峰值） | 3-4分 |
| 成交量放大（>100%峰值），且价格上涨 | 1-2分 |

**必须满足的过滤条件**：
- 收盘价 < 20日均线（短期走弱）
- 当日下跌（收盘价 < 开盘价 或 涨跌幅 < 0）
- 20日均线向上（趋势未破坏）

**数据来源**：QMT 获取日线数据，计算成交量相对位置

**计算方法**：
```python
import numpy as np
import pandas as pd

# 假设 df 包含日线数据，index为日期
df['vol_peak_10d'] = df['volume'].rolling(10).max()
df['vol_ratio'] = df['volume'] / df['vol_peak_10d']  # 成交量/峰值

df['high_10d'] = df['high'].rolling(10).max()
df['drawdown'] = (df['close'] - df['high_10d']) / df['high_10d'] * 100  # 从高点回撤

df['close_ma20'] = df['close'].rolling(20).mean()
df['ma20_slope'] = (df['close_ma20'] - df['close_ma20'].shift(5)) / df['close_ma20'].shift(5) * 100

# 缩量回调信号
signal = (df['vol_ratio'] < 0.5) & (df['drawdown'] < -10) & \
         (df['close'] < df['close_ma20']) & (df['close'] < df['open']) & \
         (df['ma20_slope'] > 0)
```

### 2. 基本面稳健度（25%权重）
**量化标准**：最近两期财报的营收和净利润同比增速

| 条件 | 得分 |
|------|------|
| 营收+净利润双增长，增速>20% | 9-10分 |
| 营收+净利润双增长，增速5-20% | 7-8分 |
| 营收或净利润其中一项增长 | 5-6分 |
| 营收+净利润均低个位数增长或持平 | 3-4分 |
| 营收或净利润出现下滑 | 1-2分 |

**数据来源**：同花顺财务数据 / 东方财富财报页面

**注意**：
- 使用 `revenue` 字段（非 `operating_revenue`，后者全为0）
- 同比用间隔4期（季度累计值）

---

### 3. 趋势健康度（25%权重）
**量化标准**：股价与20周均线的位置关系

| 条件 | 得分 |
|------|------|
| 股价在20周均线上方，且20周均线向上 | 9-10分 |
| 股价在20周均线上方，均线走平 | 7-8分 |
| 股价在20周均线附近（±3%以内） | 5-6分 |
| 股价在20周均线下方，均线走平 | 3-4分 |
| 股价在20周均线下方，且均线向下 | 1-2分 |

**数据来源**：QMT / 同花顺 / 东方财富，查看周线图20周均线位置

---

## 🧮 计算方法

### 综合评分
```
综合分 = (量价形态分 × 0.5) + (基本面分 × 0.25) + (趋势分 × 0.25)
```

### 操作决策矩阵

| 综合评分 | 建议操作 | 仓位建议 |
|----------|----------|----------|
| **8.0-10.0分** | **强烈买入** | 可重仓（50%以上） |
| **7.0-7.9分** | **正常买入** | 正常仓位（30-40%总资金） |
| **6.0-6.9分** | **轻仓试错** | 轻仓（≤20%总资金） |
| **4.0-5.9分** | **中性观望** | 不操作 |
| **0.0-3.9分** | **不买入** | 不考虑入场 |

## 🔄 使用流程

### 买入前评估（约5分钟）
1. **定量价形态**：运行脚本获取缩量回调信号 → 形态分
2. **查财报**：看最近两期营收/净利润同比增速 → 基本面分
3. **查均线**：看股价与20周均线位置 → 趋势分
4. **计算综合分**，对照决策矩阵，确定操作和仓位
5. **记录**：保存评分和决策到历史记录

---

## ⚠️ 已知局限性（v3.0诚实说明）

1. **权重未经回测验证**：50%形态权重是基于顺网科技案例的推断，需要更多样本验证
2. **趋势维度与形态有重叠**：形态信号已包含MA20方向判断，可能信息冗余
3. **缺卖出系统**：本系统只管买入，卖出纪律需另行建立
4. **样本量不足**：目前仅4个历史信号，统计显著性有限
