# 五维买入评估系统 v2.0

> 注：本系统仅用于买入决策，不考虑卖出。
> v2.0 核心升级：将主观打分改为客观量化指标，消除情绪干扰，支持历史回测。

---

## 🎯 系统目的
防止盲目追高、情绪化交易，建立可量化、可重复、可回测的买入决策框架。

---

## 📊 五个评估维度（各20%权重）

### 1. 趋势健康度（客观量化）
**量化标准**：股价与均线的位置关系

| 条件 | 得分 |
|------|------|
| 股价在20周均线上方，且20周均线向上 | 9-10分 |
| 股价在20周均线上方，均线走平 | 7-8分 |
| 股价在20周均线附近（±3%以内） | 5-6分 |
| 股价在20周均线下方，均线走平 | 3-4分 |
| 股价在20周均线下方，且均线向下 | 1-2分 |

**数据来源**：QMT / 同花顺 / 东方财富，查看周线图20周均线位置

---

### 2. 基本面稳健度（客观量化）
**量化标准**：最近两期财报的营收和净利润同比增速

| 条件 | 得分 |
|------|------|
| 营收+净利润双增长，增速>20% | 9-10分 |
| 营收+净利润双增长，增速5-20% | 7-8分 |
| 营收或净利润其中一项增长 | 5-6分 |
| 营收+净利润均低个位数增长或持平 | 3-4分 |
| 营收或净利润出现下滑 | 1-2分 |

**数据来源**：同花顺财务数据 / 东方财富财报页面

---

### 3. 情绪极端度 ⚠️ 逆向思维！（客观量化）
**量化标准**：个股换手率相对于自身历史水平的百分位

> 换手率暴增 = 市场情绪极端（恐慌或贪婪），结合价格方向判断

| 条件 | 得分 |
|------|------|
| 近5日均换手率处于过去1年**低位**（<20百分位），且股价近期下跌 | 9-10分（极度冷清=无人关注=潜在机会） |
| 近5日均换手率处于过去1年低位（20-40百分位），股价平稳 | 7-8分 |
| 近5日均换手率处于历史中位（40-60百分位） | 5-6分 |
| 近5日均换手率处于高位（60-80百分位），股价上涨 | 3-4分（追高风险） |
| 近5日均换手率处于极高位（>80百分位），股价大涨 | 1-2分（情绪过热=卖出信号） |

**数据来源**：QMT `get_history_data()` 获取换手率，计算百分位

**计算方法**：
```python
import numpy as np
# 获取过去250个交易日换手率
turnover_history = get_turnover_rate(stock, 250)
# 近5日均值
recent_avg = np.mean(turnover_history[-5:])
# 计算百分位
percentile = (np.sum(turnover_history < recent_avg) / len(turnover_history)) * 100
```

---

### 4. 价格吸引力（客观量化）
**量化标准**：当前PE（市盈率）处于过去3年历史PE的百分位

| 条件 | 得分 |
|------|------|
| 当前PE处于过去3年**低位**（<20百分位） | 9-10分 |
| 当前PE处于过去3年较低位（20-40百分位） | 7-8分 |
| 当前PE处于过去3年中位（40-60百分位） | 5-6分 |
| 当前PE处于过去3年较高位（60-80百分位） | 3-4分 |
| 当前PE处于过去3年**高位**（>80百分位） | 1-2分 |

**数据来源**：同花顺/东方财富PE历史数据，或QMT财务接口

> ⚠️ 注意：亏损股PE为负，此维度不适用，改用PB百分位替代

---

### 5. CS500环境分（保持原有逻辑）
**量化标准**：CS500每日评分转换

```
转换公式：CS500_10分制 = (CS500原始分[-10到+10] + 10) ÷ 2

示例：
- CS500 = -10分 → 环境分 0分
- CS500 =   0分 → 环境分 5分
- CS500 =  +3分 → 环境分 6.5分
- CS500 = +10分 → 环境分 10分
```

**数据来源**：每日晚间CS500评分记录

---

## 🧮 计算方法

### 综合评分
```
综合分 = (趋势健康度 + 基本面稳健度 + 情绪极端度 + 价格吸引力 + CS500环境分) ÷ 5
```

### 操作决策矩阵

| 综合评分 | 建议操作 | 仓位建议 |
|----------|----------|----------|
| **8.0-10.0分** | **强烈买入** | 可重仓（突破重仓线） |
| **7.0-7.9分** | **正常买入** | 正常仓位（30-40%总资金） |
| **6.0-6.9分** | **轻仓试错** | 轻仓（≤20%总资金） |
| **4.0-5.9分** | **中性观望** | 不操作 |
| **0.0-3.9分** | **不买入** | 不考虑入场 |

## 🔄 使用流程

### 买入前评估（约5分钟）
1. **查均线**：打开周线图，看股价与20周均线位置 → 趋势分
2. **查财报**：看最近两期营收/净利润同比增速 → 基本面分
3. **查换手率**：计算近5日均换手率的历史百分位 → 情绪分
4. **查PE**：看当前PE处于历史PE的百分位 → 价格吸引力分
5. **查CS500**：当日CS500评分转换 → 环境分
6. **计算综合分**，对照决策矩阵，确定操作和仓位
7. **记录**：保存评分和决策到历史记录

## ⚠️ 已知局限性（v2.0诚实说明）

1. **五维权重平均未经验证**：各维度各占20%是假设，未经历史回测验证哪个维度更重要
2. **维度间存在相关性**：牛市时趋势好、情绪热、PE高，三个维度会同时拉低总分，可能导致牛市里买不进去
3. **缺卖出系统**：本系统只管买入，卖出纪律需另行建立
4. **样本量不足**：目前历史验证案例极少，系统有效性尚待验证
