# 顺网科技（300113）每日评分 - 极简版

## 🎯 评分设计思路
基于CS500评分系统，针对个股特点优化：
- **技术面**：趋势、成交量、关键价位
- **基本面**：公司经营、题材热度、行业地位
- **市场情绪**：恐慌/贪婪、散户/机构态度
- **买入时机**：追高/抄底、风险/收益比

---

## 📅 每日快速评分（3分钟完成）

### 第一步：数据记录
**收盘数据**：
```

收盘价：______元
涨跌幅：______%
成交量：______亿股
```

### 第二步：技术面评分（0-10分）
**趋势判断**：
- [ ] **上涨趋势**（站上均线，放量）→ **7-10分**
- [ ] **震荡整理**（横盘，成交量平稳）→ **4-6分**  
- [ ] **下跌趋势**（跌破均线，放量跌）→ **0-3分**

**今日技术面得分**：___分

### 第三步：基本面评分（0-10分）
**公司状态**：
- [ ] **积极变化**（利好公告，业绩向好）→ **7-10分**
- [ ] **平稳发展**（无重大变化）→ **4-6分**
- [ ] **负面消息**（利空，业绩下滑）→ **0-3分**

**题材热度**：
- [ ] **热点题材**（市场追捧）→ **+2分**
- [ ] **普通题材** → **+1分**
- [ ] **冷门题材** → **0分**

**今日基本面得分**：___分

### 第四步：市场情绪评分（-10到+10分）
**恐慌/贪婪指数**：
- [ ] **极度恐慌**（暴跌，恐慌抛售）→ **-10到-7分**
- [ ] **谨慎悲观**（阴跌，担忧）→ **-6到-3分**
- [ ] **中性平衡** → **-2到+2分**
- [ ] **谨慎乐观** → **+3到+6分**
- [ ] **极度贪婪**（暴涨，狂热）→ **+7到+10分**

**今日情绪得分**：___分

### 第五步：买入时机评分（-10到+10分）
**你的买入操作评估**：
- [ ] **完美时机**（买在启动点）→ **+7到+10分**
- [ ] **较好时机**（买在相对低位）→ **+3到+6分**
- [ ] **一般时机**（买在震荡区）→ **-2到+2分**
- [ ] **较差时机**（追高买入）→ **-6到-3分**
- [ ] **最差时机**（买在暴跌前）→ **-10到-7分**

## 📊 综合评分计算

**加权计算**：
```
综合评分 = 
  技术面分×0.3 + 
  基本面分×0.3 + 
  情绪分×0.2 + 
  买入时机分×0.2
```

**最终评分区间与建议**：
```
🔥 +6分以上：强烈看涨，可加重仓
↗️ +3到+6分：温和看涨，可建仓
↔️ -2到+2分：中性震荡，持有观察
↘️ -6到-2分：温和看跌，考虑减仓
💧 -6分以下：强烈看跌，应止损
```

## 📈 今日示例（2026-03-13）

**已知数据**：
- 收盘价：26.92元
- 涨跌幅：-9.33%
- 换手率：15.80%
- 成交额：22.55亿元

**技术面**：下跌趋势，放量暴跌 → **2分**
**基本面**：公司基本面未变，但题材受关注 → **6分**  
**市场情绪**：极度恐慌，基金抛售 → **-9分**
**买入时机**：暴跌前买入，追高 → **-8分**

**综合评分**：2×0.3 + 6×0.3 + (-9)×0.2 + (-8)×0.2 = **-1.4分**
**建议**：中性偏空，不宜抄底，等待企稳
