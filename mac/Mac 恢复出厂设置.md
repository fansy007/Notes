# 恢复出厂设置
```html
苹果恢复出厂模式

在 macOS 中恢复出厂设置需要进行以下几个步骤。这将删除您的所有数据和应用程序，并将操作系统恢复到初始状态。在开始之前，请确保备份所有重要数据。

1. 备份数据：使用 Time Machine 或其他备份方法将您的数据备份到外部硬盘或云服务中。
2. 重启 Mac：点击苹果菜单（屏幕左上角的苹果标志），然后选择“重启”。
3. 进入恢复模式：在 Mac 重启过程中，立即按住 Command + R 键，直到看到苹果徽标或旋转的地球。这将进入 macOS 的恢复模式。
4. 连接到 Wi-Fi：在恢复模式下，如果需要，点击屏幕右上角的 Wi-Fi 图标，然后选择一个可用的 Wi-Fi 网络进行连接。
5. 抹掉硬盘：在 macOS 实用工具窗口中，选择“磁盘实用工具”，然后点击“继续”。选择 Mac 的内部硬盘（通常名为 “Macintosh HD” 或 “APPLE SSD”），然后点击“抹掉”按钮。确保格式设置为“APFS”或“Mac OS 扩展（日志式）”，然后点击“抹掉”。
6. 请注意，这将永久删除硬盘上的所有数据。操作前请确保数据已备份。
7. 重新安装 macOS：抹掉硬盘后，退出磁盘实用工具，回到 macOS 实用工具窗口。选择“重新安装 macOS”，然后点击“继续”。按照屏幕上的提示操作，然后等待 macOS 安装完成。这可能需要一些时间，取决于您的网络速度和设备性能。
8. 设置新系统：安装完成后，Mac 将自动重启。按照屏幕上的说明设置您的新系统。您可以在此过程中从之前的备份中迁移数据，或稍后进行迁移。

完成以上步骤后，您的 Mac 将恢复到出厂设置。
```
